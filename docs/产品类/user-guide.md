# 用户使用手册 - 银河证券证裕交易单元

## 系统简介

证裕交易单元是银河证券推出的虚拟证券交易模拟系统，帮助用户学习证券交易知识，体验交易流程。

**重要提示：本系统为虚拟交易系统，不涉及真实资金。**

## 快速开始

### 注册登录
1. 访问系统首页
2. 点击"注册"按钮
3. 填写手机号/邮箱
4. 设置密码
5. 完成注册

### 初始资金
- 新用户默认获得100万虚拟资金
- 可在个人中心查看资产

## 核心功能

### 1. 行情查看
- 实时行情：查看沪深、港股实时行情
- 自选股：添加关注的股票
- 涨跌排行：查看市场热点

### 2. 股票交易
#### 买入
1. 搜索股票代码/名称
2. 点击"买入"
3. 输入数量和价格
4. 确认交易

#### 卖出
1. 在持仓中选择股票
2. 点击"卖出"
3. 输入卖出数量
4. 确认交易

### 3. 新股申购
1. 进入"新股申购"页面
2. 查看可申购新股
3. 点击"申购"
4. 输入申购数量
5. 等待中签结果

### 4. 资产管理
- 总资产：查看总资产变化
- 持仓：查看当前持仓
- 交易记录：查看历史交易
- 资金流水：查看资金变动

### 5. 投资教育
- 投教中心：学习投资知识
- 研究报告：查看专业研报
- 投资日历：关注重要事件

## 交易规则

### 交易时间
- 模拟交易：7x24小时
- 撮合时间：实时撮合

### 交易限制
- 最小交易单位：100股（1手）
- 单笔最大：10000股
- T+1交易：当日买入次日可卖

### 手续费
- 买入：0.03%
- 卖出：0.03% + 0.1%印花税

## 常见问题

### Q: 如何充值？
A: 本系统为虚拟交易，无需充值。

### Q: 为什么无法卖出？
A: 检查是否为T+1限制，当日买入次日才能卖出。

### Q: 交易失败怎么办？
A: 检查余额是否充足，股票代码是否正确。

### Q: 如何查看收益？
A: 在"资产分析"页面查看收益曲线。
