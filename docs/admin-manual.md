# 银河证券管理系统 - 管理员手册

## 概述
本手册为银河证券管理系统（证裕交易单元）的管理员提供操作指南，涵盖7个内容管理模块：研报、投教内容、日历事件、新股、交易规则、横幅公告。

## 1. 管理员登录
1. 访问系统首页 (`/`)
2. 点击"管理员登录"或直接访问 `/admin/dashboard`
3. 使用管理员账号登录（需在Supabase profiles表中设置role='admin'）
4. 登录成功后自动跳转到管理后台仪表板

## 2. 内容管理模块

### 2.1 研报管理 (`/admin/reports`)
**功能**：管理研究报告，包括个股、行业、宏观、策略分析报告。

**字段说明**：
- **标题**：研报标题（必填）
- **作者**：研报作者（必填）
- **日期**：发布日期（必填）
- **摘要**：研报摘要（必填）
- **内容**：详细内容（可选）
- **分类**：个股/行业/宏观/策略（必填）
- **观点**：看多/中性/看空（必填）

**操作**：
- **新建**：点击"新建研报"按钮，填写表单后提交
- **编辑**：点击列表中的"编辑"按钮，修改后保存
- **删除**：点击列表中的"删除"按钮，确认后删除
- **刷新**：点击"刷新"按钮重新加载数据

### 2.2 投教内容管理 (`/admin/education`)
**功能**：管理投资者教育内容。

**字段说明**：
- **标题**：教育主题标题（必填）
- **分类**：内容分类（必填）
- **图片**：封面图片URL（可选）
- **时长**：学习时长（可选）

**操作**：同研报管理

### 2.3 日历事件管理 (`/admin/calendar`)
**功能**：管理财经日历事件。

**字段说明**：
- **标题**：事件标题（必填）
- **日期**：事件日期（必填）
- **类型**：宏观/财报/活动等（必填）
- **时间**：具体时间（可选）
- **市场**：相关市场，逗号分隔（可选）

**操作**：同研报管理

### 2.4 新股管理 (`/admin/ipos`)
**功能**：管理新股发行信息。

**字段说明**：
- **代码**：股票代码（必填）
- **名称**：股票名称（必填）
- **价格**：发行价格（必填）
- **涨跌**：价格变动（可选）
- **涨跌幅%**：百分比变动（可选）
- **市场**：CN/HK/US/BOND/FUND（必填）
- **上市日期**：上市日期（必填）
- **状态**：即将上市/已上市/已取消（必填）

**操作**：同研报管理

### 2.5 交易规则管理 (`/admin/rules`)
**功能**：配置股票交易、新股申购、大宗交易、涨停打板等各类交易的精细化规则。

**规则类型**：
- **IPO（新股申购）**：控制新股申购的各项参数
  - **中签率**：新股申购的中签概率
  - **最小申购数量**：单次申购的最低股数限制
  - **最大申购金额**：单次申购的最高金额限制
  - **每户配售上限**：每个账户最多可中签的数量
  - **锁定期(天)**：新股上市后的锁定期限制

- **BLOCK_TRADE（大宗交易）**：控制大宗交易的各项参数
  - **最小成交数量**：大宗交易的最低股数要求
  - **撮合窗口期**：大宗交易的撮合时间窗口
  - **需管理员确认**：是否需要管理员人工确认交易
  - **折价率**：大宗交易相对于市价的折扣
  - **佣金费率**：大宗交易收取的佣金费率

- **LIMIT_UP（涨停打板）**：控制涨停打板的各项参数
  - **订单优先级**：打板订单的优先级设定
  - **触发阈值**：涨停板触发的价格阈值
  - **单笔最大限额**：单笔打板订单的最大金额限制
  - **分钟频率限制**：每分钟最多下单次数
  - **日订单限制**：每日打板订单总数限制

- **GENERAL（通用交易）**：控制通用交易的风险参数
  - **交易费率**：该交易类型收取的额外手续费
  - **风险等级阈值**：触发风险控制的阈值
  - **日亏损限制**：单日最大亏损金额限制
  - **追保线**：触发追加保证金的资产比例

**操作**：编辑各规则类型的配置参数，修改后实时生效

### 2.6 横幅公告管理 (`/admin/banners`)
**功能**：管理首页横幅公告。

**字段说明**：
- **标题**：横幅标题（必填）
- **描述**：简短描述（必填）
- **图片**：横幅图片URL（必填）
- **分类**：公告分类（必填）
- **日期**：发布日期（必填）
- **内容**：详细内容（可选）
- **关联股票代码**：相关股票代码（可选）

**操作**：同研报管理

## 3. 权限管理
- **管理员**：可访问所有管理页面，进行CRUD操作
- **普通用户**：无法访问 `/admin/*` 路径，会被重定向到 `/dashboard`
- **权限验证**：通过Supabase RLS策略实现，仅role='admin'的用户可写

## 4. 常见问题

### 4.1 数据不显示
**可能原因**：
1. 数据库连接失败
2. 表无数据
3. RLS策略限制

**解决方案**：
1. 检查Supabase连接配置（`.env`文件）
2. 检查数据库表是否有数据
3. 确认用户角色为'admin'

### 4.2 操作失败
**可能原因**：
1. 网络连接问题
2. 权限不足
3. 表单验证失败

**解决方案**：
1. 检查网络连接
2. 确认登录状态和角色
3. 检查必填字段是否填写

### 4.3 图片不显示
**可能原因**：
1. 图片URL无效
2. 跨域限制
3. 图片服务器问题

**解决方案**：
1. 使用有效的图片URL
2. 确保图片支持外链
3. 使用Supabase Storage存储图片

## 5. 数据备份与恢复
- **备份**：定期导出Supabase数据库
- **恢复**：通过Supabase控制台导入数据
- **建议频率**：每日自动备份，手动备份在重大变更前

- **监控**：定期检查系统日志
- **更新**：及时更新依赖包
