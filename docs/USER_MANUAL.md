## 注册与登录

### 账号注册

**步骤**：
1. 访问官网 `https://your-domain.com`
2. 点击右上角"注册"按钮
3. 输入邮箱地址
4. 设置登录密码（8-20位，包含大小写字母和数字）
5. 输入邮箱验证码
6. 阅读并同意《用户协议》和《隐私政策》

### 账号登录

**方式一：密码登录**
1. 点击"登录"按钮
2. 输入注册邮箱和密码
3. 点击"登录"

**方式二：记住登录状态**
- 勾选"记住我"，下次访问自动登录
- 建议在个人设备上使用

**方式三：安全退出**
1. 点击右上角头像
2. 选择"退出登录"

### 忘记密码

1. 在登录页点击"忘记密码"
2. 输入注册邮箱
3. 查收邮件，点击重置链接
4. 设置新密码
5. 使用新密码登录

## 开户认证

### 开户前准备

请准备好以下材料：
- ✅ 本人有效身份证（原件）
- ✅ 本人银行卡（用于绑定资金账户）
- ✅ 手机（接收验证码）
- ✅ 良好的网络环境

### 开户流程

**步骤一：填写基本信息**
1. 登录后点击"开户"或弹窗提示
2. 填写真实姓名、身份证号
3. 选择职业、学历
4. 填写联系地址

**步骤二：身份验证**
1. 上传身份证正面照片
2. 上传身份证反面照片
3. 进行人脸识别验证
   - 保持光线充足
   - 正面面对摄像头
   - 按提示做点头、眨眼动作

**步骤三：银行卡绑定**
1. 选择开户银行
2. 输入银行卡号
3. 输入银行预留手机号
4. 输入短信验证码

**步骤四：风险测评**
1. 完成风险承受能力问卷（约10题）
2. 系统将给出风险等级：
   - C1 保守型
   - C2 稳健型
   - C3 平衡型
   - C4 成长型
   - C5 进取型

**步骤五：签署协议**
1. 阅读《证券交易委托代理协议》
2. 阅读《风险揭示书》
3. 电子签名确认
4. 提交开户申请

**步骤六：等待审核**
- 审核时间：1-2个工作日
- 审核结果将通过邮件和短信通知
- 审核通过后即可开始交易

### 开户状态查询

**路径**：我的 → 账户中心 → 开户状态

| 状态 | 说明 |
|------|------|
| 待审核 | 资料已提交，等待审核 |
| 审核中 | 工作人员正在审核 |
| 已通过 | 开户成功，可以交易 |
| 已拒绝 | 资料有误，请重新提交 |

## 交易下单

### 交易前确认

交易前请确认：
- ✅ 已完成开户并通过审核
- ✅ 账户有足够资金
- ✅ 了解交易规则和风险

### 买入股票

**路径**：交易 → 买入

**操作步骤**：
1. 输入股票代码或名称
2. 选择订单类型：
   - **限价单**：指定价格买入
   - **市价单**：以当前市场价买入
3. 输入买入数量
4. 系统自动计算预估金额（含手续费）
5. 点击"买入"
6. 确认订单信息
7. 输入交易密码
8. 提交订单

### 卖出股票

**路径**：交易 → 卖出

**操作步骤**：
1. 输入股票代码（或从持仓列表选择）
2. 选择订单类型
3. 输入卖出数量（可点击"全部"卖出全部持仓）
4. 点击"卖出"
5. 确认订单信息
6. 输入交易密码
7. 提交订单

### 订单类型说明

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| 限价单 | 以指定价格或更优价格成交 | 想控制买入/卖出价格 |
| 市价单 | 以当前最优市场价成交 | 希望快速成交 |

### 撤单

**路径**：交易 → 当日委托

1. 在"当日委托"列表中找到要撤销的订单
2. 点击"撤单"按钮
3. 确认撤单

> **注意**：已成交的订单不能撤销

### 查看订单状态

**当日委托**：查看今天下的所有订单
**历史委托**：查看历史订单记录
**当日成交**：查看今天成交的订单
**历史成交**：查看历史成交记录

**订单状态说明**：
- 待成交：订单已提交，等待成交
- 部分成交：订单部分数量已成交
- 已成交：订单全部成交
- 已撤单：订单已撤销
- 已失效：订单已过期或失败

## 资产管理

### 查看总资产

**路径**：我的 → 资产

显示信息：
- 总资产：持仓市值 + 可用资金
- 持仓市值：持有股票的市值
- 可用资金：可用于交易的资金
- 冻结资金：已下单未成交冻结的资金
- 当日盈亏：今天持仓的盈亏
- 累计盈亏：历史累计盈亏

### 持仓明细

**路径**：我的 → 持仓

显示每只股票的：
- 持仓数量
- 成本价
- 当前价
- 市值
- 盈亏金额
- 盈亏比例

**操作**：点击股票可快速跳转交易页面

### 资金流水

**路径**：我的 → 资产 → 资金流水

查看所有资金变动记录：
- 入金/出金
- 买入/卖出
- 手续费
- 分红派息
- 利息

支持按时间筛选和导出

### 充提款

**入金（充值）**：
1. 进入"我的 → 资产 → 入金"
2. 选择入金方式（银行转账/快捷支付）
3. 输入入金金额
4. 按提示完成支付
5. 资金实时到账

**出金（提款）**：
1. 进入"我的 → 资产 → 出金"
2. 选择到账银行卡
3. 输入出金金额
4. 输入交易密码
5. 提交申请
6. 资金1-2个工作日到账

> **注意**：出金前请确保持仓市值不低于最低保证金要求

## IPO新股申购

### 查看IPO日历

**路径**：IPO → IPO日历

显示即将上市的新股信息：
- 申购日期
- 上市日期
- 发行价
- 募集资金
- 申购上限

### 申购新股

**路径**：IPO → 今日申购

**申购条件**：
- 账户状态正常
- 持仓市值达到申购要求
- 账户有足够资金

**申购步骤**：
1. 在"今日申购"列表中选择新股
2. 查看新股详情和招股说明书
3. 输入申购数量（必须是申购单位的整数倍）
4. 点击"申购"
5. 确认申购信息
6. 提交申购

### 查看申购结果

**路径**：IPO → 我的申购

**状态说明**：
- 申购中：已提交申购申请
- 已中签：获得新股配额
- 未中签：未获得配额
- 已上市：新股已上市交易

**中签后操作**：
- 确保账户有足够资金缴款
- 系统将自动扣款
- 新股上市后可交易

## 常见问题

### 账户相关

**Q1: 注册后多久可以交易？**

**A**: 完成开户申请并通过审核后即可交易，通常需要1-2个工作日。

**Q2: 一个手机号可以注册几个账号？**

**A**: 一个手机号只能注册一个账号。

**Q3: 如何注销账号？**

**A**: 请联系客服提交注销申请。注销前请确保：
- 账户无持仓
- 无冻结资金
- 无未完成的订单

### 交易相关

**Q4: 交易时间是什么时候？**

**A**: 
- 港股：09:30-12:00, 13:00-16:00（香港时间）
- 美股：21:30-04:00（北京时间，夏令时）/ 22:30-05:00（冬令时）

**Q5: 下单后多久能成交？**

**A**: 市价单通常秒级成交，限价单需等待价格达到设定值。

**Q6: 为什么我的订单没有成交？**

**A**: 可能原因：
- 限价单价格未达到
- 市场流动性不足
- 非交易时间
- 订单已过期

**Q7: 交易手续费怎么计算？**

**A**: 
- 港股：交易金额的0.03%，最低3港元
- 美股：每股0.0049美元，最低0.99美元
- 具体费率请查看"我的 → 费率说明"

### 资金相关

**Q8: 入金多久到账？**

**A**: 
- 快捷支付：实时到账
- 银行转账：1-2个工作日

**Q9: 出金多久到账？**

**A**: 提交申请后1-2个工作日到账。

**Q10: 为什么无法出金？**

**A**: 可能原因：
- 持仓市值低于维持保证金
- 有未完成的交易订单
- 账户被冻结
- 出金金额超过可用资金

### 技术问题

**Q11: App/网页打不开怎么办？**

**A**: 
1. 检查网络连接
2. 清除浏览器缓存/App缓存
3. 更新到最新版本
4. 尝试更换浏览器/网络
5. 联系客服

**Q12: 行情数据有延迟吗？**

**A**: 
- 免费用户：延迟15分钟
- VIP用户：实时行情

**Q13: 收不到验证码怎么办？**

**A**: 
1. 检查垃圾邮件/短信拦截
2. 确认手机号/邮箱正确
3. 等待60秒后重新发送
4. 联系客服

### 其他问题

**Q14: 如何联系客服？**

**A**: 
- 在线客服：我的 → 在线客服（交易日09:00-18:00）
- 客服邮箱：support@yinhe.com

## 风险提示

### 投资风险提示

1. **市场风险**：证券价格受多种因素影响，存在波动风险
2. **流动性风险**：部分股票可能交易不活跃，买卖价差较大
3. **杠杆风险**：融资融券交易会放大收益和损失
4. **汇率风险**：港股、美股涉及汇率波动
5. **政策风险**：监管政策变化可能影响市场

### 交易风险提示

1. **技术风险**：网络延迟、系统故障可能导致交易失败
2. **操作风险**：误操作可能造成损失
3. **信息安全风险**：请妥善保管账号密码

### 免责声明

- 本手册仅供参考，不构成投资建议
- 投资有风险，入市需谨慎
- 请根据自身风险承受能力做出投资决策
