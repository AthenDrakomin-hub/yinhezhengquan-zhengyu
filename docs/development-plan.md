## 六、技术依赖说明

### 新增依赖项
- 无需新增依赖，使用现有技术栈实现

### 需要配合的后端接口
1. **自选股管理**: `DELETE /api/watchlist/{symbol}`, `PUT /api/watchlist/sort`
2. **板块详情**: `GET /api/sectors/{id}/stocks`, `GET /api/sectors/{id}/flow`
3. **交易结果**: 现有接口已支持
4. **意见反馈**: `POST /api/feedback`, `GET /api/feedback/history`

### 涉及的共享组件
- `TradeResultModal` - 交易结果弹窗
- `WatchlistManager` - 自选股管理器
- `PositionContext` - 持仓上下文
