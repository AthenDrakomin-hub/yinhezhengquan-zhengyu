## 系统概述

### 管理端功能模块

| 模块 | 功能说明 |
|------|---------|
| 用户管理 | 查看用户信息、审核开户申请、管理用户状态 |

### 支持的管理员角色

| 角色 | 权限范围 |
|------|---------|
| 超级管理员 | 所有权限，包括创建其他管理员 |

| 运营人员 | 内容管理、工单管理、数据统计（只读） |

## 登录与权限

### 管理员登录

1. 访问管理端登录页面：`https://your-domain.com/auth/login`
2. 输入管理员邮箱和密码
3. 输入双因素认证码（TOTP）
4. 点击登录进入管理后台

> **注意**：管理员账号需要在数据库中手动设置 `admin_level` 字段为 `admin` 或 `super_admin`

### 权限说明

#### 超级管理员权限
- 创建/编辑/删除其他管理员账号
- 修改系统核心配置
- 查看所有操作日志
- 执行敏感操作（如强制平仓）

#### 普通管理员权限
- 管理普通用户
- 处理交易相关操作
- 管理内容和工单
- 查看统计数据

## 用户管理

### 查看用户列表

**路径**：管理后台 → 用户管理

**功能说明**：
- 查看所有注册用户列表
- 支持按用户名、邮箱、状态筛选
- 查看用户资产、持仓、交易记录

**操作步骤**：
1. 点击左侧菜单"用户管理"
2. 使用搜索框输入关键词筛选用户
3. 点击用户行查看详情

### 审核开户申请

**路径**：管理后台 → 用户管理 → 待审核

**审核流程**：
1. 查看待审核用户列表
2. 点击用户查看开户资料
   - 身份证信息
   - 人脸识别结果
   - 风险测评结果
   - 电子签名
3. 点击"通过"或"拒绝"
4. 填写审核意见（拒绝时必填）

### 管理用户状态

**可设置的用户状态**：

| 状态 | 说明 | 影响 |
|------|------|------|
| 正常 | 用户可以正常交易 | 无限制 |
| 冻结 | 暂停交易功能 | 不能下单，可以查看 |
| 封禁 | 完全禁用账号 | 无法登录 |
| 待审核 | 开户申请审核中 | 不能交易 |

**操作方法**：
1. 在用户详情页点击"编辑状态"
2. 选择新状态
3. 填写状态变更原因
4. 点击确认

## 交易管理

### 监控交易订单

**路径**：管理后台 → 交易管理 → 订单列表

**功能说明**：
- 实时查看所有用户交易订单
- 支持按订单状态、时间范围筛选
- 查看订单详情和成交记录

**订单状态说明**：

| 状态 | 说明 |
|------|------|
| PENDING | 待成交 |
| FILLED | 已成交 |
| CANCELLED | 已撤单 |
| FAILED | 失败 |

### 批量委托

**路径**：管理后台 → 交易管理 → 批量委托

**使用场景**：
- 系统维护时批量处理订单
- 特殊行情下统一操作

**操作步骤**：
1. 上传批量委托文件（CSV格式）
2. 预览委托列表
3. 确认无误后点击"执行批量委托"

**CSV格式要求**：
```csv
user_id,symbol,type,side,quantity,price
uuid,000001,BUY,LIMIT,100,10.50
```

### 强制平仓

**路径**：管理后台 → 交易管理 → 持仓监控

**使用场景**：
- 用户保证金不足
- 风控要求紧急平仓

**操作步骤**：
1. 在持仓列表中找到需要平仓的用户
2. 点击"强制平仓"
3. 选择平仓方式：
   - 市价平仓（立即以市场价卖出）
   - 限价平仓（设置指定价格）
4. 填写平仓原因
5. 二次确认后执行

> **警告**：强制平仓是不可逆操作，请谨慎使用！

## 内容管理

### 管理首页Banner

**路径**：管理后台 → 内容管理 → Banner管理

**功能说明**：
- 添加/编辑/删除首页轮播图
- 设置跳转链接和显示顺序
- 控制上线/下线状态

**添加Banner步骤**：
1. 点击"新增Banner"
2. 上传图片（建议尺寸：1920x600px）
3. 填写标题和描述
4. 设置跳转链接（可选）
5. 设置显示顺序（数字越小越靠前）
6. 选择上线状态
7. 点击保存

### 教育中心管理

**路径**：管理后台 → 内容管理 → 教育中心

**功能模块**：
- 课程分类管理
- 课程内容编辑
- 学习进度查看

**添加课程步骤**：
1. 选择或创建课程分类
2. 点击"新增课程"
3. 填写课程信息：
   - 课程标题
   - 封面图片
   - 课程简介
   - 难度等级（入门/进阶/高级）
4. 添加课程内容（支持Markdown）
5. 设置课程状态（草稿/发布）
6. 点击保存

### 公告管理

**路径**：管理后台 → 内容管理 → 公告

**公告类型**：
- 系统公告（所有用户可见）
- 活动公告（特定用户群体）
- 维护通知（系统维护前发送）

## 工单管理

### 处理用户工单

**路径**：管理后台 → 工单管理

**工单状态**：
- 待处理：新提交的工单
- 处理中：正在跟进
- 已解决：问题已解决
- 已关闭：工单关闭

**处理流程**：
1. 查看待处理工单列表
2. 点击工单查看详情
3. 阅读用户问题和历史记录
4. 在回复框输入解决方案
5. 如需转交，选择转交对象
6. 点击"提交回复"
7. 问题解决后标记为"已解决"

### 工单分类

| 分类 | 说明 |
|------|------|
| 账户问题 | 登录、注册、密码相关 |
| 交易问题 | 下单、成交、资金相关 |
| 技术问题 | 系统Bug、功能异常 |
| 投诉建议 | 用户投诉、产品建议 |
| 其他 | 不属于以上分类 |

## 系统设置

### 管理员账号管理

**路径**：管理后台 → 系统设置 → 管理员

> **仅限超级管理员操作**

**添加管理员**：
1. 点击"添加管理员"
2. 输入邮箱、用户名
3. 设置初始密码
4. 选择角色（管理员/运营人员）
5. 点击保存

**修改权限**：
1. 在管理员列表点击"编辑"
2. 修改角色或状态
3. 点击保存

### 系统参数配置

**路径**：管理后台 → 系统设置 → 参数配置

**可配置项**：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 交易时间 | 允许交易的时间段 | 09:30-11:30, 13:00-15:00 |
| 最小下单金额 | 单笔交易最低金额 | 100元 |
| 最大下单金额 | 单笔交易最高金额 | 100万 |
| 风控阈值 | 触发风控的持仓比例 | 80% |

### IP白名单

**路径**：管理后台 → 系统设置 → 安全设置

**功能**：限制管理员登录的IP地址

**配置方法**：
1. 开启"IP白名单"开关
2. 输入允许的IP地址（多个用逗号分隔）
3. 点击保存

## 常见问题

### Q1: 如何重置管理员密码？

**A**: 
1. 超级管理员在"系统设置 → 管理员"中找到该账号
2. 点击"重置密码"
3. 系统生成随机密码并发送邮件
4. 管理员首次登录需修改密码

### Q2: 用户投诉交易异常如何处理？

**A**:
1. 在工单系统查看投诉详情
2. 进入"交易管理 → 订单查询"核实订单
3. 查看系统日志确认是否存在异常
4. 联系用户说明情况
5. 如确属系统问题，提交技术部门处理

### Q3: 如何导出用户数据？

**A**:
1. 进入"用户管理"
2. 点击"导出数据"
3. 选择导出字段和时间范围
4. 选择格式（CSV/Excel）
5. 点击导出，下载文件

### Q4: 系统出现大面积故障怎么办？

**A**:
1. 立即在"公告管理"发布维护通知
2. 暂停交易功能（系统设置 → 交易开关）
3. 联系技术团队紧急处理
4. 在"操作日志"中记录故障时间
5. 恢复后发布恢复公告

### Q5: 如何查看操作日志？

**A**:
1. 进入"系统设置 → 操作日志"
2. 支持按管理员、时间、操作类型筛选
3. 点击日志行查看详情

## 安全规范

### 管理员账号安全

1. **定期更换密码**（建议90天）
2. **开启双因素认证**（必须）
3. **不在公共网络登录管理后台**
4. **离开座位时锁定屏幕**
5. **不向他人透露管理员账号**

### 敏感操作规范

以下操作需要二次确认：
- 强制平仓
- 封禁用户账号
- 删除管理员账号
- 修改系统核心配置

### 数据安全

1. 用户数据不得导出到个人设备
2. 不得在公共渠道讨论用户敏感信息
3. 定期备份重要数据
4. 离职时立即禁用账号
