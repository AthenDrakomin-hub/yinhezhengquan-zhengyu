# 项目背景信息

## 项目概述
**银河证券管理系统 - 证裕交易单元**是一个工业级虚拟证券交易平台，提供完整的证券交易业务闭环模拟系统。本项目为银河证券内部开发的虚拟交易系统，不涉及真实资金交易，主要用于培训、演示和系统验证。

## 核心定位
- **虚拟交易模拟**：完整的行情同步、交易下单、清算对账全流程
- **工业级标准**：采用金融行业最佳实践和合规标准
- **教育演示工具**：用于证券交易知识培训和系统操作演示

## 业务价值
1. **培训价值**：为证券从业人员提供真实的交易环境模拟
2. **技术验证**：验证交易系统架构和技术方案的可行性
3. **合规演示**：展示证券交易系统的合规性和安全性设计
4. **产品原型**：作为未来真实交易系统的技术原型

## 关键特性
- ✅ 完整业务闭环：行情 → 开户 → 交易 → 撮合 → 清算 → 审计
- ✅ 真实数据源：集成新浪财经、东方财富等实时行情数据
- ✅ 合规安全：RLS权限隔离、操作审计、数据加密传输
- ✅ 智能功能：批量操作、智能推荐、性能监控、热度统计
- ✅ 工业级UI：Tailwind CSS 4.2铝感工业风设计

## 技术架构
- **前端**：React 19 + TypeScript 5.7 + Vite 6.4
- **后端**：Supabase (PostgreSQL + Edge Functions)
- **部署**：Vercel平台
- **UI框架**：Tailwind CSS 4.2 + Framer Motion 12.34
- **状态管理**：React Hooks + Context API

## 用户角色
1. **普通用户**：进行虚拟证券交易操作
2. **管理员**：管理系统配置、用户管理、交易监控
3. **合规人员**：审计交易记录、监控合规性

## 开发原则
1. **安全第一**：所有交易均为虚拟，不涉及真实资金
2. **合规优先**：遵循证券行业监管要求
3. **性能优化**：确保系统响应速度和稳定性
4. **用户体验**：提供直观易用的交易界面

## 注意事项
⚠️ **重要声明**：本系统为虚拟交易系统，所有交易均为模拟操作，不涉及真实资金流动。系统主要用于培训和演示目的，不应作为真实投资决策依据。
